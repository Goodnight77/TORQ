# TORQ Demo Data

Data used to drive the scripted demo and back the Fault-to-Fix pipeline.

- **manuals/** — OEM equipment manuals sourced from IBM AssetOpsBench; the AI
  agent retrieves from these (via MCP) to diagnose faults.
- **history/** — historical failure/repair logs from IBM AssetOpsBench used as
  repair-history context.
- **scenarios/** — seed fault scenarios that drive the scripted demo run.
