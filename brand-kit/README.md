# TORQ Brand Kit

## Logo

`logo.svg` - wordmark with a gear mark. Use on dark backgrounds. Keep clear space
around it equal to the height of the "T". Do not recolor the wordmark or stretch it.

## Name and tagline

- Name: **TORQ** (all caps).
- Product: TORQ Fault-to-Fix Engine.
- Tagline: **From fault code to fixed.**
- One-liner: Autonomous machine diagnosis and repair dispatch.

## Color palette

| Role | Hex |
|------|-----|
| Background (dark) | `#0F1720` |
| Surface / card | `#111C28` |
| Primary (TORQ green) | `#3FB950` |
| Text | `#E6EDF3` |
| Muted text | `#8B98A5` |
| Alert / fault | `#F85149` |

Primary green is the accent for the mark, key numbers, and calls to action. Use it
sparingly against the dark background for contrast.

## Typography

- Primary: Segoe UI / system sans-serif (Arial fallback).
- Arabic: Amiri (used in generated work-order PDFs).
- Headings bold; body regular; numbers can use the green accent.

## Voice

Direct, quantified, confident. Lead with numbers. No hype. "Turns the most
expensive 30 minutes in manufacturing into 30 seconds."
